# 🔐 Secure Coding Review

This project was developed as part of my Cyber Security Internship at CodeAlpha.

The objective of this project is to identify security vulnerabilities in insecure Python authentication code and implement secure coding practices to improve application security.

## 📌 Vulnerabilities Identified

* Hardcoded credentials
* Weak passwords
* Plain text password storage
* Insecure authentication logic

## 🛡️ Security Improvements Implemented

* SHA-256 password hashing
* Strong password implementation
* Improved authentication process
* Secure credential handling

## 🧠 Learning Outcomes

Through this project, I learned:

* The importance of secure software development
* How attackers exploit weak authentication systems
* How password hashing protects sensitive data
* Best practices for secure coding in Python

## 🛠️ Technologies Used

* Python
* hashlib (SHA-256)
